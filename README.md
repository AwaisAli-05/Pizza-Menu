 # 🍕 Fast React Pizza Co.

A modern and responsive pizza menu application built with **React.js**. The application displays a restaurant's menu with pizza details, prices, availability status, and opening hours using reusable React components.

## ✨ Features

- 🍕 Display a dynamic pizza menu
- 📋 Reusable React components
- 📱 Responsive layout
- 💲 Shows pizza prices
- 🚫 Indicates sold-out items
- 🕒 Displays restaurant opening status
- ⚡ Fast and lightweight React application

---

## 🛠️ Built With

- React.js
- JavaScript (ES6+)
- JSX
- CSS3
- HTML5

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/fast-react-pizza.git
```

### Navigate to the project

```bash
cd fast-react-pizza
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```
